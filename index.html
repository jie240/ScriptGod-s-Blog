ScriptGod编程语言介绍

🎯 核心定位
可商用的轻量级自动化工具 - 采用 MIT 协议，允许开发者将其集成到自己的产品中并闭源售卖。

✨ 主要功能模块

  1. 网络爬虫
    get - HTTP GET 请求

header - 设置请求头

json - 解析 JSON 响应

xpath <表达式> - HTML/XML 解析

save <文件路径> - 保存响应内容

  1. 数据处理
    CSV 操作: csv_read / csv_write

Excel 操作: excel_read / excel_write（基于 openpyxl）

支持数据导入导出和格式转换

  1. 编程逻辑
    变量赋值（自动类型识别）

if 条件判断（支持 > < == !=）

for 循环（数字范围循环）

func 自定义函数定义与调用

多线程定时任务（timer）

  1. GUI 交互
    gui_msg - 消息弹窗

gui_input - 输入对话框

可快速构建简单的桌面交互界面

  1. 文件与系统
    read / write - 文本文件读写

list - 目录列表

sleep - 延时控制（毫秒）

🔧 技术架构
text
词法分析器 (Tokenizer) → 虚拟机 (VM) → 执行引擎
词法分析: 将脚本代码转换为 token 流，支持字符串、数字、标识符和运算符

虚拟机: 解释执行 token，管理变量作用域和函数表

扩展性: 基于 Python 实现，可轻松添加新功能

💼 商业价值
低代码特性: 业务人员经过简单培训即可编写自动化脚本

MIT 协议: 可自由集成到商业产品中，无需开源

零依赖运行: 封装后用户无需安装 Python 环境

场景广泛:

数据采集与清洗

定时报表生成

办公自动化（Excel/CSV 处理）

GUI 自动化工具

📝 示例脚本
text

爬取网页并保存

header User-Agent “Mozilla/5.0”
get “https://example.com“
save “page.html”

Excel 数据处理

excel_read “data.xlsx”
csv_write “output.csv”

定时任务

func mytask
print “执行定时任务”
}

timer 5 mytask

条件判断

name = “FlowScript”
if name == “FlowScript”
gui_msg “欢迎使用 FlowScript！”
}
🚀 使用方式
bash

运行脚本

python flowscript.py run script.fs
🎨 设计亮点
类自然语言语法: 关键字直观易懂（get、save、print等）

即时反馈: 支持交互式消息提示

容错处理: 异常捕获机制，避免脚本崩溃

轻量级: 核心代码仅 200+ 行，易于维护和定制

这个引擎特别适合需要快速实现自动化但又不想深入编程语言的场景，可作为企业内部工具链的脚本平台，或作为低代码产品的底层引擎。
